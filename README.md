# RMIT-Cloud-Computing-2610-A2
Assignment 2 - AWS Cloud System Development

AWS Cloud Music Subscription System

This repository contains the source code, infrastructure scripts, and documentation for Assessment 2 of the RMIT Cloud Computing (2610) course. The project is a cloud-based web application for music subscriptions, deployed across three distinct backend architectures using Amazon Web Services (AWS).
